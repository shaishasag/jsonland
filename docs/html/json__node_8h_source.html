928 // the only place where there could be a value without string representation is when a number value is assigned

931 // is called - requiring the number to be converted again from text to binary form. Another inefficiantiancly could
932 // occur in the same json_node is repeatadly assigned number values. The compramise is to store the number as it was
933 // given (text when parsed, binary when assigned) and treat it accodingly.
