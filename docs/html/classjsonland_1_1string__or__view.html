size_t jsonland::string_or_view::allocation_size ( ) const
inlinenoexcept
Returns
Estimation of amount of heap memory allocated by this object, not including the object itself.
If holding std::string, will try to take into account "small string optimization", in which case 0 will be reported.
If holding std::string_view, 0 will be reported.
